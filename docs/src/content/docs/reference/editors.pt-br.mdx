Os seguintes editores podem ser usados para criar, compilar e visualizar fluxos de trabalho agentic. Alguns são ferramentas integradas mantidas juntamente com o gh-aw; outros são projetos criados pela comunidade.

### Compiler Playground

Um playground interativo baseado em navegador **experimental** que executa o compilador gh-aw inteiramente no navegador usando [WebAssembly](/gh-aw/reference/wasm-compilation/). Ele demonstra como usar a build WASM do compilador diretamente em uma página da web e mostra como compilar fluxos de trabalho no navegador usando o motor de execução baseado em WASM.

- https://github.github.com/gh-aw/editor/

## Editores da comunidade

> [!NOTE]
> Editores da comunidade são criados e mantidos por contribuidores independentes. Eles não são oficialmente suportados pelo projeto gh-aw.

### Gerador de Prompt Agentic

Uma ferramenta baseada na web para gerar e editar prompts de fluxo de trabalho agentic. Ele fornece uma interface interativa para ajudar a criar prompts de fluxo de trabalho para fluxos de trabalho agentic.

- https://ashleywolf.github.io/agentic-prompt-generator/

![Captura de tela do Gerador de Prompt Agentic](https://github.com/github/gh-aw/blob/assets/Weekly-Editors-Health-Check/e6a9c35f3724c642f733bc2e0f2009d8478584998269e89649089e3f3c8ab2b4.png?raw=true)

### Editor Gráfico de Fluxo de Trabalho

Um editor visual e gráfico de fluxo de trabalho que fornece uma interface de usuário mais rica para editar fluxos de trabalho agentic. Em vez de trabalhar diretamente com markdown e YAML, este editor foca em uma experiência de edição mais interativa e visual.

- https://mossaka.github.io/gh-aw-editor-visualizer/
