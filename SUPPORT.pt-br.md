# Suporte

## Como abrir tickets e obter ajuda

Este projeto utiliza os tickets do GitHub para acompanhar bugs e solicitações de recursos. Por favor, pesquise os tickets existentes antes de abrir novos para evitar duplicatas. Para novos tickets, registre seu bug ou solicitação de recurso como um novo ticket.

Para obter ajuda ou esclarecer dúvidas sobre o uso deste projeto, por favor, abra um ticket.

## Política de Suporte do GitHub

O suporte para este projeto limita-se aos recursos listados acima.
